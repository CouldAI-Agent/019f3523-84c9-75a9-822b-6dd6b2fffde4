# Polytechnic And University Management System

A comprehensive Flutter application for managing polytechnic and university activities, featuring separate portals for students and teachers.

## Features
- **Student & Teacher Login**: Secure role-based access.
- **Attendance**: Track and view daily attendance records.
- **Result**: Access academic results and grades.
- **Routine**: View class schedules and timetables.
- **Notice**: Stay updated with the latest announcements.
- **Assignment Upload**: Submit and manage assignments efficiently.

## Tech Stack
- Flutter (Cross-platform: iOS, Android, Web, Desktop)
- Dart

## Setup Instructions
1. Clone the repository.
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run` to launch the app on your preferred device or emulator.
